Как запустить бота на Windows:

1. Распакуй архив.
2. Открой папку juicy_shop_bot.
3. Запусти start.bat двойным кликом.

Если запускаешь через CMD:
cd app
venv\Scripts\activate
python main.py

Важно:
- .env содержит токен, ADMIN_ID и MANAGER_USERNAME. Не отправляй этот файл другим людям.
- Если меняешь токен/админа/менеджера — редактируй .env и app/.env.
